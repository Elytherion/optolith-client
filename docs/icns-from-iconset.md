# `.icns` file from `.iconset` folder

```sh
iconutil -c icns <PATH_TO_FOLDER>
```

Make sure all of the following files are available. The tool is **case-sensitive**.

```
icon_16.png
icon_16@2x.png
icon_32.png
icon_32@2x.png
icon_128.png
icon_128@2x.png
icon_256.png
icon_256@2x.png
icon_512.png
icon_512@2x.png
```
